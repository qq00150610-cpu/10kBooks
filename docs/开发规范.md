## 1. 代码规范

### 1.1 Java 编码规范

#### 1.1.1 命名规范

| 类型 | 规范 | 示例 |
|------|------|------|
| **类名** | UpperCamelCase | `UserService`, `BookController` |
| **方法名** | lowerCamelCase | `getUserById`, `saveBook` |
| **变量名** | lowerCamelCase | `userName`, `bookList` |
| **常量** | UPPER_SNAKE_CASE | `MAX_RETRY_COUNT`, `DEFAULT_PAGE_SIZE` |
| **枚举** | UpperCamelCase | `OrderStatus`, `PayChannel` |
| **包名** | 全小写 | `com.tenkbooks.user` |

#### 1.1.2 代码示例

```java
// ✅ 推荐的类定义
public class UserServiceImpl implements UserService {
    
    // 常量定义
    private static final int MAX_QUERY_SIZE = 1000;
    private static final Logger log = LoggerFactory.getLogger(UserServiceImpl.class);
    
    // 成员变量
    private final UserRepository userRepository;
    private final PasswordEncoder passwordEncoder;
    
    // 构造函数注入
    @Autowired
    public UserServiceImpl(UserRepository userRepository, 
                          PasswordEncoder passwordEncoder) {
        this.userRepository = userRepository;
        this.passwordEncoder = passwordEncoder;
    }
    
    // 方法命名
    public UserDTO getUserById(Long userId) {
        log.debug("查询用户信息, userId: {}", userId);
        User user = userRepository.findById(userId)
            .orElseThrow(() -> new UserNotFoundException(userId));
        return convertToDTO(user);
    }
    
    public List<UserDTO> listUsers(UserQuery query) {
        // 前置校验
        Assert.notNull(query, "查询参数不能为空");
        
        // 业务逻辑
        Page<User> page = userRepository.findAll(query.toSpec(), PageRequest.of(
            query.getPage() - 1, 
            Math.min(query.getSize(), MAX_QUERY_SIZE)
        ));
        
        return page.getContent().stream()
            .map(this::convertToDTO)
            .collect(Collectors.toList());
    }
}
```

#### 1.2.2 代码示例

```typescript
// types/user.ts
export interface User {
  id: string;
  nickname: string;
  avatar?: string;
  email: string;
  status: UserStatus;
  createdAt: Date;
}

export enum UserStatus {
  Active = 1,
  Inactive = 0,
  Banned = -1
}

// services/user.service.ts
import { Injectable } from '@angular/core';
import { HttpClient } from '@angular/common/http';
import { Observable } from 'rxjs';
import { map } from 'rxjs/operators';
import { User, UserDTO } from '@/types/user';

@Injectable({
  providedIn: 'root'
})
export class UserService {
  private readonly API_URL = '/api/v1/users';
  
  constructor(private http: HttpClient) {}
  
  getUserById(id: string): Observable<UserDTO> {
    return this.http.get<User>(`${this.API_URL}/${id}`).pipe(
      map(user => this.toDTO(user))
    );
  }
  
  getUserList(params: UserQueryParams): Observable<PageResult<UserDTO>> {
    return this.http.get<PageResult<User>>(`${this.API_URL}`, { params }).pipe(
      map(page => ({
        ...page,
        data: page.data.map(this.toDTO)
      }))
    );
  }
  
  private toDTO(user: User): UserDTO {
    return {
      id: user.id,
      nickname: user.nickname,
      avatar: user.avatar || '/assets/default-avatar.png',
      email: this.maskEmail(user.email)
    };
  }
  
  private maskEmail(email: string): string {
    const [name, domain] = email.split('@');
    return `${name.slice(0, 2)}***@${domain}`;
  }
}
```

### 1.3 SQL 规范

```sql
-- ✅ 推荐：表名和字段名使用小写下划线
SELECT user_id, nickname, created_at 
FROM user 
WHERE status = 1;

-- ✅ 推荐：使用具体的字段名
SELECT id, username, email FROM user WHERE id = ?;

-- ✅ 推荐：使用 LIMIT
SELECT * FROM book ORDER BY created_at DESC LIMIT 10;

-- ✅ 推荐：使用预编译语句
PREPARE stmt FROM 'SELECT * FROM user WHERE id = ?';
EXECUTE stmt USING @user_id;

-- ❌ 禁止：SELECT *
SELECT * FROM user;

-- ❌ 禁止：LIKE 模糊匹配以通配符开头
SELECT * FROM book WHERE title LIKE '%小说';

-- ❌ 禁止：在 WHERE 中使用函数
SELECT * FROM user WHERE DATE(created_at) = '2024-01-01';
```

## 2. Git 提交规范

### 2.1 Commit 格式

```
<type>(<scope>): <subject>

<body>

<footer>
```

### 2.2 Type 类型

| 类型 | 说明 | 示例 |
|------|------|------|
| feat | 新功能 | feat(user): 添加第三方登录 |
| fix | Bug 修复 | fix(book): 修复章节加载慢的问题 |
| docs | 文档更新 | docs: 更新 API 文档 |
| style | 代码格式 | style: 格式化代码 |
| refactor | 重构 | refactor(order): 重构支付流程 |
| perf | 性能优化 | perf: 优化首页加载速度 |
| test | 测试相关 | test: 添加单元测试 |
| chore | 构建/工具 | chore: 升级依赖版本 |

### 2.3 Scope 范围

| 模块 | 说明 |
|------|------|
| user | 用户模块 |

| order | 订单模块 |
| payment | 支付模块 |
| social | 社交模块 |
| admin | 管理后台 |

### 5.2 单元测试规范

```java
// ✅ 推荐的单元测试
@ExtendWith(MockitoExtension.class)
class UserServiceTest {
    
    @Mock
    private UserRepository userRepository;
    
    @InjectMocks
    private UserServiceImpl userService;
    
    @Test
    void testGetUserById_Success() {
        // Given
        Long userId = 1L;
        User user = User.builder()
            .id(userId)
            .nickname("test")
            .email("test@example.com")
            .build();
        when(userRepository.findById(userId)).thenReturn(Optional.of(user));
        
        // When
        UserDTO result = userService.getUserById(userId);
        
        // Then
        assertThat(result.getId()).isEqualTo(userId);
        assertThat(result.getNickname()).isEqualTo("test");
        verify(userRepository, times(1)).findById(userId);
    }
    
    @Test
    void testGetUserById_NotFound() {
        // Given
        Long userId = 999L;
        when(userRepository.findById(userId)).thenReturn(Optional.empty());
        
        // When & Then
        assertThatThrownBy(() -> userService.getUserById(userId))
            .isInstanceOf(UserNotFoundException.class)
            .hasMessageContaining("999");
    }
}
```
