# 作者系统

## 模块概述

作者系统为创作者提供完整的创作、发布、变现一站式服务。涵盖作者入驻、作品管理、章节发布、数据分析、收益结算等功能。

### 2.1 入驻流程

```
┌─────────────────────────────────────────────────────────────────────────┐
│                           作者入驻流程                                    │
└─────────────────────────────────────────────────────────────────────────┘

  ┌─────────┐       ┌─────────┐       ┌─────────┐       ┌─────────┐

       │                 │                 │                 │
       │ 1.申请作者         │                 │                 │
       │─────────────────▶│                 │                 │
       │                 │ 2.提交资料         │                 │
       │                 │─────────────────▶│                 │
       │                 │                 │ 3.审核资料        │
       │                 │                 │─────────────────▶│

```

### 2.2 入驻条件

| 类型 | 条件 | 说明 |
|------|------|------|
| **普通作者** | 年满 18 岁，身份证认证 | 可发布免费/付费作品 |
| **签约作者** | 普通作者 + 优质作品 + 平台邀请 | 享有保底收益、专属运营 |
| **白金作者** | 签约作者 + 多部热门作品 + 平台认定 | 全平台资源倾斜 |

### 2.3 申请资料

| 资料项 | 说明 | 必填 |
|--------|------|------|
| penName | 笔名 | 是 |
| bio | 作者简介 | 是 |
| idCard | 身份证号 | 是 |
| idCardFront | 身份证正面 | 是 |
| idCardBack | 身份证背面 | 是 |
| bankName | 开户银行 | 是 |
| bankAccount | 银行卡号 | 是 |

## 3. 作品管理

### 3.1 创建作品

**请求参数**

```json
{
  "title": "星际穿越：宇宙的奥秘",
  "subtitle": "一场跨越星际的冒险之旅",
  "categoryId": "cat_001",
  "tags": ["科幻", "冒险", "热血"],
  "coverUrl": "https://cdn.10kbooks.com/covers/xxx.jpg",
  "description": "这是一个关于星际冒险的故事...",
  "isPaid": true,
  "price": 9.99,
  "currency": "CNY"
}
```

### 3.2 作品状态流转

```
┌─────────────────────────────────────────────────────────────────────────┐
│                           作品状态流转                                   │
└─────────────────────────────────────────────────────────────────────────┘

  ┌─────────┐
  │  草稿   │ (draft)
  └────┬────┘
       │ 发布
       ▼
  ┌─────────┐     ┌─────────┐
  │ 审核中  │────▶│ 已发布  │◀───┐
  └────┬────┘ 拒绝 │─────────┘    │ 重新发布
       │           │             │
       │           │─────────────┘
       │           │
       ▼           ▼
  ┌─────────┐  ┌─────────┐
  │ 被禁用  │  │ 已完结  │
  └─────────┘  └─────────┘
```

### 3.3 作品信息

| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 作品ID |
| title | VARCHAR(200) | 书名 |
| coverUrl | VARCHAR(500) | 封面 |
| categoryId | INT | 分类ID |
| tags | VARCHAR[] | 标签 |
| description | TEXT | 简介 |
| wordCount | INT | 字数 |
| chapterCount | INT | 章节数 |
| viewCount | BIGINT | 阅读量 |
| recommendCount | INT | 推荐票 |
| collectCount | INT | 收藏数 |
| status | TINYINT | 状态 |

## 7. 收益管理

### 7.1 收益构成

| 来源 | 分成比例 | 说明 |
|------|---------|------|
| 订阅分成 | 50%-70% | VIP 订阅收入分成 |
| 付费购买 | 60%-80% | 单章/整书购买收入 |
| 打赏分成 | 70% | 读者打赏收入 |
| 全勤奖励 | ¥500-2000 | 月全勤奖励 |
| 推荐奖励 | ¥1000-5000 | 获得平台推荐 |

### 7.2 提现规则

| 项目 | 说明 |
|------|------|
| 最低提现金额 | ¥100 |
| 提现周期 | 每月 1-10 日 |
| 到账时间 | 申请后 3-7 工作日 |
| 手续费 | 免手续费 |

### 7.3 收益计算示例

```
作品订阅收益计算：

总收入 = VIP 订阅收入 + 付费购买收入 + 打赏收入

作者分成 = (VIP订阅收入 × 分成比例) + (付费购买收入 × 分成比例) + (打赏收入 × 70%)

示例：
- 本月 VIP 订阅收入：¥10,000
- 付费购买收入：¥5,000
- 打赏收入：¥2,000
- 分成比例：65%

作者收益 = (10,000 × 65%) + (5,000 × 70%) + (2,000 × 70%)
         = 6,500 + 3,500 + 1,400
         = ¥11,400
```

## 8. API 接口

### 8.1 作者入驻

| 接口 | 方法 | 说明 |
|------|------|------|
| `/v1/author/apply` | POST | 申请成为作者 |
| `/v1/author/status` | GET | 获取申请状态 |
| `/v1/author/profile` | GET/PUT | 作者主页信息 |

### 8.2 作品管理

| 接口 | 方法 | 说明 |
|------|------|------|
| `/v1/author/books` | GET | 作品列表 |
| `/v1/author/books` | POST | 创建作品 |
| `/v1/author/books/{bookId}` | GET/PUT | 作品详情/编辑 |
| `/v1/author/books/{bookId}` | DELETE | 删除作品 |
| `/v1/author/books/{bookId}/cover` | POST | 上传封面 |

### 8.3 章节管理

| 接口 | 方法 | 说明 |
|------|------|------|
| `/v1/author/books/{bookId}/chapters` | GET | 章节列表 |
| `/v1/author/books/{bookId}/chapters` | POST | 创建章节 |
| `/v1/author/books/{bookId}/chapters/{chapterId}` | GET/PUT | 章节详情/编辑 |
| `/v1/author/books/{bookId}/chapters/{chapterId}` | DELETE | 删除章节 |
| `/v1/author/books/{bookId}/chapters/{chapterId}/publish` | POST | 发布章节 |

### 8.4 数据分析

| 接口 | 方法 | 说明 |
|------|------|------|
| `/v1/author/stats/overview` | GET | 数据概览 |
| `/v1/author/stats/{bookId}` | GET | 作品数据 |
| `/v1/author/stats/trend` | GET | 趋势数据 |
| `/v1/author/stats/readers` | GET | 读者画像 |

### 8.5 收益管理

| 接口 | 方法 | 说明 |
|------|------|------|
| `/v1/author/income/overview` | GET | 收益概览 |

| `/v1/author/income/withdraw` | POST | 申请提现 |
| `/v1/author/income/history` | GET | 提现记录 |
