# 阅读器

## 模块概述

阅读器是平台的核心用户体验模块，提供沉浸式的电子书阅读体验。支持多格式阅读、个性化设置、离线缓存、听书等功能。

### 2.2 格式解析

```javascript
// 阅读器引擎架构
class ReaderEngine {
  // 格式解析器
  parsers = {
    epub: new EPUBMarser(),
    pdf: new PDFParser(),
    txt: new TXTParser(),
    mobi: new MOBIParser()
  };

  // 解析书籍
  async parse(file) {
    const ext = getFileExtension(file.name);
    const parser = this.parsers[ext];
    if (!parser) throw new Error('Unsupported format');
    return parser.parse(file);
  }

  // 渲染内容
  render(book, options) {
    // 统一的内容渲染接口
  }
}
```

## 8. 离线缓存

### 8.1 缓存策略

| 内容 | 缓存方式 | 说明 |
|------|---------|------|
| 当前章节 | 内存 + 本地 | 实时阅读 |
| 前后章节 | 本地存储 | 预加载 |
| 整本书 | 可选 | 用户主动下载 |
| 书籍信息 | 本地 + 服务端 | 长期缓存 |

### 8.2 存储管理

```javascript
// 离线管理
class OfflineManager {
  // 下载整本书
  async downloadBook(bookId) {
    const chapters = await this.getChapterList(bookId);
    for (const chapter of chapters) {
      await this.cacheChapter(chapter);
    }
  }

  // 清理缓存
  async clearCache(bookId) {
    await this.storage.remove(`book_${bookId}`);
  }

  // 存储空间管理
  async getStorageInfo() {
    return {
      used: await this.storage.used(),
      total: this.storage.quota(),
      cacheList: await this.getCacheList()
    };
  }
}
```

## 10. 性能优化

### 10.1 加载优化

| 策略 | 说明 |
|------|------|
| 章节预加载 | 当前章节加载时，预加载下一章 |
| 虚拟列表 | 长章节使用虚拟滚动 |
| 图片懒加载 | 图片进入视口才加载 |
| 内容分块 | 大章节分块加载 |

### 10.2 渲染优化

| 策略 | 说明 |
|------|------|
| 分页渲染 | 每次只渲染当前页 |
| Canvas 绘制 | 复杂排版使用 Canvas |
| 缓存 DOM | 复用章节 DOM 节点 |
| GPU 加速 | 动画使用 GPU 渲染 |
