# 多语言支持

## 模块概述

多语言支持是平台国际化(I18n)的基础能力，为全球用户提供本地化体验。涵盖界面翻译、内容翻译、地区适配等功能。

## 3. 界面国际化

### 3.1 前端实现

```javascript
// i18n 配置
import { createI18n } from 'vue-i18n';
import enUS from '@/locales/en-US';
import zhCN from '@/locales/zh-CN';
import jaJP from '@/locales/ja-JP';

const i18n = createI18n({
  legacy: false,
  locale: localStorage.getItem('locale') || 'zh-CN',
  fallbackLocale: 'en-US',
  messages: {
    'en-US': enUS,
    'zh-CN': zhCN,
    'ja-JP': jaJP
  },
  // 数字格式化
  numberFormats: {
    'en-US': {
      currency: { style: 'currency', currency: 'USD' }
    },
    'zh-CN': {
      currency: { style: 'currency', currency: 'CNY' }
    },
    'ja-JP': {
      currency: { style: 'currency', currency: 'JPY' }
    }
  },
  // 日期格式化
  datetimeFormats: {
    'en-US': {
      short: { year: 'numeric', month: 'short', day: 'numeric' },
      long: { year: 'numeric', month: 'long', day: 'numeric', weekday: 'long' }
    },
    'zh-CN': {
      short: { year: 'numeric', month: 'short', day: 'numeric' },
      long: { year: 'numeric', month: 'long', day: 'numeric', weekday: 'long' }
    }
  }
});

export default i18n;
```

## 6. RTL 语言支持

### 6.1 RTL 语言

阿拉伯语、希伯来语等从右到左书写的语言需要特殊处理。

```css
/* RTL 适配 */
[dir="rtl"] {
  /* 翻转方向 */
  direction: rtl;
  text-align: right;
}

[dir="rtl"] .navbar {
  flex-direction: row-reverse;
}

[dir="rtl"] .icon-arrow {
  transform: scaleX(-1);
}
```

```javascript
// RTL 检测与设置
function setupRTL(locale) {
  const rtlLocales = ['ar-SA', 'he-IL', 'fa-IR'];
  const isRTL = rtlLocales.includes(locale);
  
  document.documentElement.dir = isRTL ? 'rtl' : 'ltr';
  document.documentElement.lang = locale;
}
```

## 7. 语言检测

### 7.1 检测策略

```javascript
// 语言检测优先级
function detectLocale() {
  // 1. URL 参数
  const urlLocale = new URLSearchParams(window.location.search).get('lang');
  if (urlLocale && isValidLocale(urlLocale)) {
    return urlLocale;
  }
  
  // 2. 本地存储
  const storedLocale = localStorage.getItem('locale');
  if (storedLocale && isValidLocale(storedLocale)) {
    return storedLocale;
  }
  
  // 3. 浏览器语言
  const browserLocale = navigator.language;
  if (isValidLocale(browserLocale)) {
    return browserLocale;
  }
  
  // 4. 默认语言
  return 'zh-CN';
}
```

### 7.2 语言切换

```javascript
// 切换语言
async function switchLocale(newLocale) {
  // 1. 保存偏好
  localStorage.setItem('locale', newLocale);
  
  // 2. 加载翻译资源
  const messages = await loadMessages(newLocale);
  
  // 3. 更新 i18n 实例
  i18n.global.locale.value = newLocale;
  
  // 4. 更新 HTML 属性
  document.documentElement.lang = newLocale;
  
  // 5. 触发 RTL 切换 (如需要)
  setupRTL(newLocale);
  
  // 6. 刷新页面 (可选)
  // window.location.reload();
}
```

## 8. API 接口

### 8.1 国际化接口

| 接口 | 方法 | 说明 |
|------|------|------|
| `/v1/i18n/locales` | GET | 获取支持的语言列表 |
| `/v1/i18n/messages/{locale}` | GET | 获取语言包 |
| `/v1/i18n/user-locale` | GET/PUT | 获取/设置用户语言偏好 |

### 8.2 内容翻译接口

| 接口 | 方法 | 说明 |
|------|------|------|
| `/v1/books/{bookId}/translations` | GET | 获取书籍翻译版本 |
| `/v1/books/{bookId}/translate` | POST | 申请翻译 |
| `/v1/translations/{transId}/status` | GET | 翻译进度查询 |
