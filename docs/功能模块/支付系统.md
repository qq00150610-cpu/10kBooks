# 支付系统

## 模块概述

支付系统为平台提供完整的支付能力，包括虚拟货币充值、书籍购买、VIP订阅、打赏等功能，支持多种支付渠道，保障交易安全。

## 7. 退款流程

### 7.1 退款规则

| 商品类型 | 退款条件 | 退款比例 |
|---------|---------|---------|
| 付费书籍 | 7天内未阅读 >30% | 100% |
| VIP订阅 | 7天内未使用 | 100% |
| 付费章节 | 不支持退款 | 0% |
| 充值余额 | 支持退款 | 100% (收手续费) |

### 7.2 退款流程

```
用户申请退款 ──▶ 系统审核 ──▶ 符合条件 ──▶ 原路退回 ──▶ 完成
                     │                  │
                     │ 不符合条件        │ 余额不足
                     ▼                  ▼
                拒绝退款            部分退款
                     │                  │
                     ▼                  ▼
                通知用户            人工处理
```

## 8. 风控系统

### 8.1 风控规则

| 规则 | 说明 | 处理 |
|------|------|------|
| 频繁充值 | 1小时内充值>5次 | 验证码确认 |
| 大额交易 | 单笔>1000元 | 人工审核 |
| 异常IP | 多账号同一IP | 账号关联审查 |
| 设备异常 | 频繁更换设备 | 安全验证 |

### 8.2 监控指标

| 指标 | 阈值 | 告警 |
|------|------|------|
| 支付成功率 | < 95% | ⚠️ 警告 |
| 退款率 | > 5% | ⚠️ 警告 |
| 欺诈率 | > 0.1% | 🔴 严重 |
| 客诉率 | > 1% | ⚠️ 警告 |

## 9. API 接口

### 9.1 订单接口

| 接口 | 方法 | 说明 |
|------|------|------|
| `/v1/orders` | GET | 订单列表 |
| `/v1/orders/books` | POST | 购买书籍 |
| `/v1/orders/chapters` | POST | 购买章节 |
| `/v1/orders/{orderNo}` | GET | 订单详情 |
| `/v1/orders/{orderNo}/cancel` | POST | 取消订单 |
| `/v1/orders/{orderNo}/refund` | POST | 申请退款 |

### 9.2 支付接口

| 接口 | 方法 | 说明 |
|------|------|------|
| `/v1/pay/create` | POST | 创建支付 |
| `/v1/pay/alipay` | POST | 支付宝支付 |
| `/v1/pay/wechat` | POST | 微信支付 |
| `/v1/pay/callback/alipay` | POST | 支付宝回调 |
| `/v1/pay/callback/wechat` | POST | 微信回调 |

### 9.3 钱包接口

| 接口 | 方法 | 说明 |
|------|------|------|
| `/v1/wallet` | GET | 钱包信息 |
| `/v1/wallet/recharge` | POST | 充值 |
| `/v1/wallet/withdraw` | POST | 提现 |
| `/v1/wallet/logs` | GET | 交易明细 |

### 9.4 VIP 接口

| 接口 | 方法 | 说明 |
|------|------|------|
| `/v1/vip/plans` | GET | VIP套餐列表 |
| `/v1/vip/subscribe` | POST | 开通VIP |

| `/v1/vip/cancel` | POST | 取消订阅 |
